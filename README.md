# SpringBoot
## 참고 서적 📖
코딩 자율학습 스프링 부트 3 자바 백엔드 개발 입문
## 개발 환경 💻
- IDE : VSCode
- 개발 언어 : Java 17
- 개발 환경 : SpringBoot 3.21 , gradle , jpa(Spring Data JPA), mustache
- 데이터베이스 : H2
- 형상관리 툴 : GitHub
  
## 주요 기능 📍
- 스프링 부트 3 설정
- 간단한 REST API 구현 (CRUD)
- Gradle을 사용한 빌드 구성
- Mustache 템플릿을 활용한 프론트엔드 연동
- 데이터베이스 연동
